# SwordNine

#### 介绍
SwordNine是一个用于快速进行开发的代码生成平台以及数据建模平台。它可以生成任何语言的重复代码，包括一些简单的数据验证代码，例如：非空、引用完整性、正则、多元化业务唯一属性校验。

#### 软件架构
框架及数据库：.net core3.1+Mysql8<br/>
ORM：同时支持Dapper和SqlSugar框架<br/>
缓存：支持内存以及redis，可达到无缝切换
